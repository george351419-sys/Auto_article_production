# 全平台起标题专业技能

> 标题是内容的第一道过滤器。再好的内容，标题不行 = 没有读者。
>
> 不同平台有不同的阅读场景和流量推荐逻辑，标题必须按平台差异化生成。

## 一、标题心理学基础（通用）

### 1.1 六种底层驱动力

每一种成功标题都至少激活以下一个心理机制：

| 驱动力 | 原理 | 标题示例 |
|--------|------|----------|
| **好奇缺口** | 知道了一点但不够，必须点击填补 | "我做了这件事3年，才发现从一开始就错了" |
| **价值锚定** | 明确告诉读者能获得什么 | "读完这5本书，你的认知水平超过90%的人" |
| **身份认同** | 让目标读者感觉到「这是写给我的」 | "INFP在职场中的3个隐藏天赋" |
| **损失厌恶** | 害怕错过、害怕落后、害怕吃亏 | "2024年最后一个月，这3个趋势你还不知道？" |
| **社交货币** | 分享这个标题能显得我有品味/有见识 | "一篇论文让我重新理解了AI对齐" |
| **反常识冲击** | 违背直觉、推翻常识、制造认知冲突 | "勤奋正在毁掉你" |

### 1.2 好标题的4个检验标准

写好标题后，逐一检查：

1. **瞬间理解**：读者在1秒内知道文章在讲什么、对她有什么用？
2. **对象明确**：目标读者看完觉得「这就是写给我的」？
3. **信息增量**：标题本身已经提供了有价值的信息，而不是空洞表态？
4. **点击驱动**：标题有明确的行动提示（点开看看、收藏备用）？

### 1.3 标题质量检查清单

```
□ 标题在信息流/列表页是否一眼就能看懂？
□ 标题是否包含至少1个具体数字/时间/场景？
□ 标题是否避免了「的」「了」「在」「是」等虚词堆砌？
□ 标题是否避免了陈词滥调（「不得不看」「深度好文」「惊人发现」）？
□ 标题是否有足够的差异化——换了其他平台也能用吗？
□ 标题是否真实反映了文章内容（不造谣不夸大不标题党）？
□ 标题是否有情绪锚点（好奇/焦虑/期待/认同）？
□ 标题是否太长会被截断（视平台而定）？
```

## 二、7种标题核心公式

### 公式1：数字+结果承诺

**结构**：「[数字][动作/场景]，[具体结果]」

通用版：「3个方法，帮你把文章打开率提升300%」
认知版：「我研究了1000个副业案例，发现了5个规律」
反常识版：「30岁后我放弃的7件事，反而让我活得更好」

**关键**：数字必须具体（3/5/7/10/21/30/100），不能是「几个/很多」

### 公式2：痛点+解决方案

**结构**：「别再[错误做法]了，[正确做法]才是关键」

通用版：「别再盲目日更了，真正该做的是这3件事」
认知版：「别再刷碎片信息了，深度学习只需要这一步」
行动版：「做自媒体1年没起色？试试这个调整」

**关键**：痛点必须是读者真实感受到的，不是你想出来的

### 公式3：反常识/认知颠覆

**结构**：「你以为[常识认知]？其实[真相]」

温和版：「你以为鸡汤没用？其实你一直读的是假鸡汤」
激进版：「努力不是美德，选择才是」
角度版：「职场最大的陷阱，不是不够努力，而是太努力」

**关键**：反常识必须有理有据，不能为反而反

### 公式4：悬念+价值承诺

**结构**：「[主体]的秘密/真相/逻辑，[交付承诺]」

通用版：「小红书起号的底层逻辑，终于有人讲透了」
认知版：「为什么高手都在用这个模型？今天一次讲清楚」
深度版：「字节跳动的增长飞轮，藏在这3张图里」

**关键**：悬念必须有「谜底」，不能只挖坑不填坑

### 公式5：权威背书+具体洞察

**结构**：「[背书主体][做了什么事]，[核心发现]」

通用版：「我研究了100个头部账号，发现爆款的共同点是…」
案例版：「张一鸣的内部讲话里，藏着普通人逆袭的秘密」
数据版：「分析了5000条爆款标题，我们发现了这个规律」

**关键**：背书要真实、具体，不要空洞的「专家说」

### 公式6：时间窗口+紧迫感

**结构**：「[时间限定]，[如果不行动就会怎么样]」

通用版：「2025年已经过了一半，这3个趋势你抓住了吗？」
紧迫版：「AI时代正在淘汰这5种人，希望不是你」
短期版：「高考后30天：决定你未来4年的3个选择」

**关键**：紧迫感必须有真实的时间窗口，不能人为制造焦虑

### 公式7：清单体/攻略体

**结构**：「[动作]必备[数量][品类]/[对象]必看[内容]」

通用版：「自媒体新手的28个避坑指南（建议收藏）」
实用版：「给职场新人的7条建议：越早看到越好」
清单版：「2025年必读的10本书：提升认知和收入」

**关键**：清单要有实质内容，不能是凑数清单

## 三、平台差异化起标题策略

### 3.1 公众号标题策略

**阅读场景**：朋友圈/订阅列表，标题需要在中性环境中产生点击冲动
**推荐逻辑**：社交传播（朋友在看+朋友圈分享）> 算法推荐
**核心诉求**：制造社交价值——分享这个标题显得有品味/有见识/有用

| 策略 | 公式 | 示例 |
|------|------|------|
| 数字锚定+结果感 | [数字]+[动作]+[结果] | "3个判断，帮你选出最适合你的副业" |
| 痛点直击+方案 | 别再[错误]，应该[正确] | "别再刷短视频了，真正该做的是这件事" |
| 认知升级 | 我花了[时间]，发现了[认知] | "我研究了50个35岁转型成功的人，发现了3个规律" |
| 反常识/反差 | 你以为[常识]？其实[真相] | "你以为焦虑是坏事？其实它是你最好的信号" |
| 悬念+价值承诺 | [主题]的秘密，终于有人讲透了 | "公众号起号的真相，一篇讲清楚" |
| 场景+身份标签 | [身份]+[场景]+[解决方案] | "普通人做自媒体，最该先做的一件事" |

**长度**：18-25字。太短信息不够，太长朋友圈会被截断。
**要点**：标题必须制造社交传播价值——读者转发=在向朋友传递价值。
**要避免的**：纯粹事实陈述（「今天天气很好」）、空泛表态（「人生感悟」）、无信息量的感叹（「太厉害了」）。

### 3.2 小红书标题策略

**阅读场景**：信息流瀑布式浏览，需要标题在0.5秒内抓住注意力
**推荐逻辑**：互动率（点赞收藏评论）> 点击率 > 阅读完成率
**核心诉求**：制造情绪冲动——看到就想点、就想收藏、就想评论

| 策略 | 公式 | 示例 |
|------|------|------|
| 情绪词前置 | 绝了/救命/后悔没早看到/真香 + [内容] | "后悔没早点看到的3本书，看完直接清醒" |
| 身份标签 | [身份]+必看/必备/专属 + [内容] | "INFJ们请进：这3个习惯正在消耗你" |
| 结果前置 | 坚持[动作][时间]后，我[结果] | "坚持写日记100天后，我的焦虑消失了" |
| 清单体 | [数量]个[主题]+推荐/分享/必备 | "分享5个让效率翻倍的工作习惯" |
| 对比反差 | 以前我[X]，直到试了[Y] | "以前我总加班，直到学会了这个时间管理法" |
| 悬念/钩子 | 我发现了[惊喜/震撼]，原因是… | "我发现了一个颠覆认知的思维模型，原来我们…" |
| 避坑/省钱 | [主题]+避坑指南/别再踩坑了 | "自媒体新手最容易踩的5个坑，你中了几个？" |

**长度**：12-20字，首屏完整显示。
**要点**：可以在标题中加入适当emoji点缀，但不要堆砌。
**要避免的**：官方书面语、长复合句、没有情绪的词（「关于」「对于」「针对」）。

### 3.3 今日头条标题策略

**阅读场景**：信息流混合推荐，标题需要在广告轰炸中突围
**推荐逻辑**：点击率（CTR）> 阅读完成率 > 互动率
**核心诉求**：制造信息增量和点击紧迫感

| 策略 | 公式 | 示例 |
|------|------|------|
| 信息增量 | 最新/重磅/刚刚/突发 + [信息] | "重磅：2025年高考政策出现3个重大变化" |
| 数字+悬念 | [数字]个[信号/趋势]+表明[结论] | "5个信号表明，AI正在重新定义白领工作" |
| 争议/对比 | [A]还是[B]？答案出乎意料 | "考公还是考研？3年后的差距让人震惊" |
| 干货前置 | 深度好文/干货/一文读懂+[主题] | "深度好文：普通人如何抓住AI时代的机会" |
| 紧迫感 | 赶紧/马上/别错过 + [信息] | "马上收藏：2025年高考志愿填报避坑大全" |
| 实用价值 | [动作]+指南/教程/方法 | "自媒体从0到1起号全攻略（建议收藏）" |
| 结果导向 | 做了[动作]后，[结果] | "坚持副业一年后，我的工资翻了3倍" |

**长度**：20-28字，信息流缩略图可读。
**要点**：标题信息密度更高，多用「的」「了」「是」来增加标题的完成感。
**要避免的**：纯情绪无信息、太长被截断、标题与内容不符。

## 四、高级标题技巧

### 4.1 标题卖点拆解（写标题前先问自己）

文章最核心的**3个卖点**是什么？
1. _______
2. _______
3. _______

然后给每个卖点出3个不同的标题方向，至少9个标题待选。

### 4.2 标题角度矩阵

同一个核心观点，换5个角度出标题：

| 角度 | 公式 | 示例（观点：张一鸣的延迟满足） |
|------|------|------|
| 结果角度 | 做了X，获得Y | "坚持延迟满足10年，张一鸣做对了什么" |
| 过程角度 | 如何/怎么做到X | "张一鸣的延迟满足：不是忍，而是选择" |
| 身份角度 | [身份]应该怎么做 | "创业者必看：张一鸣的延迟满足思维" |
| 危机角度 | 不做X会怎样 | "没有延迟满足的能力，做不成任何大事" |
| 对比角度 | X vs Y，差距在哪 | "张一鸣和普通CEO的差距：延迟满足" |

### 4.3 标题的「三层检验」

写完后用三层标尺检验：

**第一层：看得懂吗？**
→ 一个路人扫一眼，0.5秒内理解文章主题

**第二层：想点吗？**
→ 读者有没有「我必须点开看看」的冲动

**第三层：会分享吗？**
→ 标题有没有社交货币——转发这个标题=在传递什么价值

### 4.4 标题A/B测试策略

同内容多标题发布后的追踪指标：

| 平台 | 测试方法 | 关键指标 | 分析周期 |
|------|---------|----------|----------|
| 公众号 | 不同标题不同群发 | 打开率/分享率 | 24小时 |
| 小红书 | 不同封面+标题组合 | 点击率/互动率 | 6-12小时 |
| 今日头条 | 首发标题 vs 修改标题 | CTR/阅读完成率 | 2-4小时 |

## 五、标题SOP工作流

生成标题时，按以下流程执行：

### Step 1: 内容理解

通读文章，提取核心观点、目标受众、传播价值、情绪基调。

### Step 2: 卖点拆解

从文章中提取3个核心卖点：
- 信息卖点：读者看完能获得什么认知？
- 情绪卖点：读者看完会产生什么情绪（焦虑/期待/共鸣/反常识）？
- 行动卖点：读者看完会去做什么？

### Step 3: 按平台选择公式

根据目标平台选择2-3个最匹配的标题公式：
- 公众号：数字结果感/认知升级/反常识
- 小红书：情绪前置/身份标签/结果前置
- 今日头条：信息增量/干货前置/紧迫感

### Step 4: 批量生成

每个公式生成1-2个标题，每个平台至少3个不同角度的标题。

### Step 5: 质量检查

用标题质量检查清单逐条检验，淘汰不合格标题。

### Step 6: 分层输出

输出按平台分组的标题组，每组标注：
- 推荐标题（最适合发布的）
- 备用标题（次优选择）
- 备选标题（第三选择）

## 六、输出格式

标题大师的输出必须按以下格式组织：

```
标题优化报告

## 公众号（推荐标题）
1. [推荐标题1] - 公式：[公式名称] - 卖点：[卖点说明]
2. [推荐标题2] - 公式：[公式名称] - 卖点：[卖点说明]
3. [推荐标题3] - 公式：[公式名称] - 卖点：[卖点说明]

## 公众号（备用标题）
4. [备用标题1]
5. [备用标题2]

## 小红书（推荐标题）
1. [推荐标题1] - 公式：[公式名称] - 卖点：[卖点说明]
...

## 小红书（备用标题）
...

## 今日头条（推荐标题）
...

## 今日头条（备用标题）
...

## 标题质量检查
- 所有推荐标题均已通过三层检验（看得懂/想点/会分享）
- 标题真实反映文章内容（不夸大不造谣）
- 各平台标题长度控制在推荐范围内
- 标题之间保持了差异化卖点方向，避免同一角度重复
```

> 注意：标题大师输出的是标题优化建议，不覆盖章上线的元数据。
> 章上线的元数据包（keywords/tags/summary/pinned等）仍保留，
> 标题大师只负责生成更优质的标题组并写入发布包。
> 在流水线中，标题大师的输出将替换章上线元数据中的标题部分。
