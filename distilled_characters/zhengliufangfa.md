# 市面主流人物思维蒸馏方法调研 +「蒸笼阁」融合版 Skill 设计方案
基于当前开源圈、学术圈、实操派的成熟方案，我先系统盘点各家核心方法论，再整合优势输出一套可直接落地为 Skill 的完整蒸馏体系——「蒸笼阁」，兼顾**工程化流水线、学术级精细度、落地保真度**三大维度。

---

## 一、市面主流蒸馏方法核心调研
### 1. 女娲.skill：工业级开源蒸馏标杆（当前最成熟的落地方案）
这是2026年国内爆火的开源人物蒸馏框架，GitHub星标超8k，核心是把“蒸馏人物思维”做成了标准化流水线，最终输出可直接调用的 `.skill` 文件。
- **核心方法论：五阶蒸馏流水线**
  1. **并行调研**：6个独立 Agent 同时从6个维度采集素材——著作系统思考、长对话即兴表达、表达DNA细节、他者批评视角、关键决策记录、人生时间线，确保信息多维覆盖，避免单一视角偏差。
  2. **框架提炼**：通过**三重验证规则**筛选核心心智模型：跨领域重复出现、对新问题具备预测力、非普适性观点（体现人物独特性），三者同时满足才会被收录。
  3. **结构化构建**：输出标准化 Skill 文件，包含5层核心内容：表达DNA、心智模型、决策启发式、反模式与价值观底线、诚实边界。
  4. **质量验证**：三板斧校验——已知信息准确性校验、边界感校验（不超纲瞎编）、风格化一致性校验。
  5. **双 Agent 精炼**：A 生成回答、B 按人物标准打分修正，反复迭代至符合人设。
- **可借鉴亮点**：多Agent并行采料、三重验证去伪、五层输出结构、最终产物直接是可执行的Skill格式，工程化程度最高。

### 2. The Relic Condition：学术派九模块精细蒸馏
来自新南威尔士大学的学术研究，仅通过公开论文就能蒸馏出学术大牛的完整思维模式，盲测中达到大学高级讲师水平，是目前精细度最高的认知蒸馏框架。
- **核心方法论：九模块认知架构**，从边界到行为完整定义人物思维：
  | 模块 | 核心作用 | 蒸馏价值 |
  |------|----------|----------|
  | Scope 边界 | 定义人物的领域范围、时间分析尺度 | 明确能力边界，避免全知全能失真 |
  | Activation 激活 | 触发人物深度分析的条件，什么问题会启动严肃思考 | 还原人物的注意力触发点 |
  | Ontological 本体重构 | 人物面对问题时，先如何定义、拆解分析对象 | 对应“观察世界的独特视角”核心 |
  | Procedural 程序操作 | 人物固定的分析步骤、论证流程 | 对应可复用的思维模型 |
  | Evaluative 评估阈值 | 人物判断强弱、好坏、成败的量化/定性标准 | 还原决策的底层标尺 |
  | Intertextual 互文引用 | 人物习惯引用的理论、案例、权威来源 | 还原认知体系的底层支撑 |
  | Rhetorical 修辞节奏 | 语言组织方式、论证节奏、比喻偏好 | 对应表达DNA |
  | Refusal 拒答模式 | 人物回避哪些问题、如何拒绝表态 | 还原价值观底线与认知盲区 |
  | Provenance 演化溯源 | 人物观点随时间的变化路径 | 避免静态认知，还原成长逻辑 |
- **可借鉴亮点**：首次把“拒答模式、认知演化、本体重构”纳入蒸馏体系，完美补充了“独特视角”和“边界保真”两个核心维度，解决了普通蒸馏“只会说正确的废话、没有人物棱角”的问题。

### 3. 民间实操派：素材保真交叉验证法
以“姜胡说蒸馏”等实战案例为代表，核心诉求是**避免语录拼接，确保思维真实可复用**，是落地性最强的平民方法论。
- **核心方法论：四步保真蒸馏**
  1. **一手素材优先原则**：本人著作>本人直播/访谈逐字稿>第三方转述，素材置信度分级，低置信度内容仅作参考不进核心模型。
  2. **多场景交叉验证**：同一观点/公式在2种以上不同场景（书、直播、连麦、文章）重复出现，才判定为底层认知，单次言论直接剔除。
  3. **模型带局限性说明**：每个提炼出的思维模型必须标注“适用场景”和“失效场景”，拒绝造神。
  4. **已知问题回测**：用3个目标人物公开回答过的问题测试蒸馏结果，逻辑一致性达标才算合格。
- **可借鉴亮点**：极度务实的保真思路，用最简单的交叉规则解决了“把临时观点当底层认知”的最大失真坑。

### 4. 大模型工程蒸馏：可迁移的工程化思维
AI领域的知识蒸馏（Knowledge Distillation）虽然针对模型，但核心逻辑完全可迁移到人物思维蒸馏：
- **思维链（CoT）提取**：不仅提取结论，更要提取“从问题到结论的完整思考步骤”，对应蒸馏中“不仅要知道他怎么选，更要知道他怎么想”。
- **数据清洗流水线**：格式过滤、长度过滤、正确性验证、重复去重，对应素材预处理的去噪标准。
- **师生对齐校验**：学生模型输出与教师模型对齐，对应“反向推演验证”——用蒸馏出的模型预判人物行为，和真实行为对比校准。

## 二、「蒸笼阁」融合版蒸馏方法论（整合各家优势）
### 核心设计理念
以“**分层蒸馏+交叉保真+边界闭环**”为核心，从上到下萃取「表达风格→思维工具→决策规则→世界观视角→边界演化」五层结构，全程嵌入三重验证，最终输出可直接封装为 Skill 的标准化思维包。

### 六步标准蒸馏流水线（可直接写入 Skill 工作流）
#### 第1步：多源素材归档与置信度分级
融合女娲6维度采集+实操派一手优先原则
1. **素材分类归档**（6大类，对应6个采集维度）
   - 系统输出：著作、书籍、长文、专栏（置信度最高）
   - 即兴表达：访谈、播客、直播逐字稿、对话记录
   - 决策行为：关键选择、公开决策、行动轨迹
   - 碎片表达：社交媒体、短评、语录、公开回复
   - 他者视角：第三方评价、批评、合作者访谈
   - 时间线：人生关键节点、认知转变事件
2. **清洗去噪**
   - 剔除：客套话、情绪性吐槽、单次偶然言论、第三方过度解读
   - 标注：每条素材标注置信度（S/A/B/C级），仅S/A级可进入核心模型萃取

#### 第2步：表层萃取：思维链三元组提取
融合大模型CoT思路，从原始素材中拆解出标准化单元
- 对每一段有效素材，提取三元组：**【问题场景 → 完整思考路径 → 最终结论/决策】**
- 同步统计：高频词汇、常用比喻、口头禅、固定句式，生成表达DNA初稿

#### 第3步：中层蒸馏：思维模型与决策规则
这是“思维模型”的核心萃取层，融合Relic的程序/评估模块+女娲的决策启发式
1. **分析框架萃取**：人物拆解问题的固定维度、分类方法、常用模型（比如永远从“成本-收益-风险”三维分析）
2. **归因逻辑萃取**：遇到问题时的归因倾向（内/外归因、单/多层归因、长/短期视角）
3. **推理范式萃取**：归纳/演绎、线性/系统思维、具象/抽象偏好
4. **决策启发式萃取**：优先级排序规则、取舍原则、容错观念、判断好坏的明确标准
5. **评估阈值萃取**：人物判断“合格/优秀/失败”的具体标尺，比如乔布斯的“3秒懂价值”标准

#### 第4步：深层蒸馏：独特视角与世界观
这是“观察世界的独特视角”核心层，融合Relic本体重构+深层世界观挖掘
1. **本体论视角**：人物面对事物时，第一关注点是什么？（别人看结果他看过程，别人看收益他看风险）
2. **底层假设**：对人性、世界、时间的默认信念（人性本善/趋利？世界是确定/混沌的？）
3. **价值排序**：自由、利益、道义、创新、安稳等核心价值的优先级
4. **认知盲区**：人物默认“哪些事不可能”“哪些规则不能碰”的隐形边界

#### 第5步：边界补全：反模式与认知演化
融合Relic拒答模式+女娲诚实边界，解决失真问题
1. **思维反模式**：人物明确反对、批判的思考方式和行为
2. **价值观底线**：绝对不会触碰的原则、绝对不会做的事
3. **能力边界**：人物不擅长、公开承认不懂的领域
4. **认知演化**：不同人生阶段的观点变化，标注认知转变的关键节点

#### 第6步：三重验证与结构化封装
融合三家验证方法，确保蒸馏结果保真
1. **交叉一致性验证**：同一认知点，必须在≥2类不同素材、≥2个不同场景中同时出现，才纳入核心结论
2. **已知问题回测验证**：选取3个目标人物公开回答过的问题，用蒸馏出的模型推导答案，匹配度≥80%才算合格
3. **边界合规验证**：测试超纲问题，确认能主动承认局限，不强行输出
4. **最终封装**：按五层结构整理为标准化思维包，每个模型/观点都标注素材来源佐证

### 五层最终输出结构（标准产物模板）
| 层级 | 核心内容 | 对应价值 |
|------|----------|----------|
| 表达DNA层 | 语言风格、修辞习惯、常用比喻、口头禅、论证节奏 | 还原说话质感，提升对话沉浸感 |
| 思维工具层 | 核心思维模型、分析框架、推理范式、常用理论 | 可复用的思考方法论 |
| 决策规则层 | 判断标准、优先级排序、取舍原则、容错观、评估阈值 | 还原决策逻辑，可用于预判选择 |
| 世界观层 | 价值排序、底层假设、独特观察视角、核心信念 | 还原“看世界的独特眼光”，体现人物独特性 |
| 边界演化层 | 反模式、价值观底线、能力边界、认知演化路径 | 保真防失真，明确Skill的能力范围 |

## 三、「蒸笼阁」Skill 落地方案
### 1. Skill 基础定位
- **Skill 名称**：蒸笼阁 · 人物思维蒸馏工坊
- **核心能力**：接收用户上传的文本材料（著作、访谈、文章、语录等），自动化完成人物思维模型与独特视角的蒸馏，输出标准化、可直接调用的人物思维技能包。
- **适用场景**：蒸馏名人思维、复刻前辈经验、分析人物认知、打造个人数字分身。

### 2. Skill 核心工作流（内置执行步骤）
1. **素材导入**：支持粘贴文本、上传文档，自动识别并分类素材类型
2. **自动清洗**：自动去噪、标注置信度、提取思维链三元组
3. **分层蒸馏**：按五层结构自动萃取核心认知，生成初稿
4. **交叉验证**：自动执行三重校验，标记存疑点，输出置信度
5. **产物生成**：输出完整蒸馏报告 + 可直接使用的人物思维Prompt + 思维模型清单
6. **迭代优化**：支持用户补充素材、调整重点，重新蒸馏优化

### 3. 核心输出产物
1. 《人物思维蒸馏总报告》：五层结构完整呈现，每条结论附素材佐证
2. 《人物思维调用Prompt》：可直接复制给大模型使用，让AI以该人物的思维方式回答问题
3. 《核心思维模型手册》：每个模型配定义、应用场景、使用步骤、局限性
4. 《独特视角分析说明》：专门拆解该人物和普通人的认知差异，解释其视角的独特性

### 4. 内置校验规则（Skill 硬约束）
- 禁止无依据脑补：所有结论必须有对应素材支撑，标注来源
- 禁止全知全能：必须明确标注能力边界，超纲问题主动说明
- 禁止静态化：若素材包含时间跨度，必须体现认知演化
- 禁止语录堆砌：所有金句必须归属于对应思维模型，不得单独罗列

# 提炼他人思维模型&世界观的完整方法论
核心逻辑：**从表层内容→行为逻辑→底层认知→世界观假设**逐层萃取，像蒸馏一样“去杂质、留内核”，分为**4大阶段+落地工具+实操步骤**，可直接套用在文本、访谈、作品、言行等各类材料上。

## 一、先明确核心定义（锚定萃取目标）
1. **思维模型**：对方解决问题、推理判断、决策行动的**固定思考框架、规则、流程、工具**（偏“方法论、做事逻辑”）。
2. **观察视角**：对方看待人/事/世界的**立场、关注点、价值偏好、归因方式、底层信念**（偏“世界观、认知底色”）。
3. 蒸馏原则：**剥离情绪、场景、表象，保留可复用的认知规律**。

---

## 二、全流程方法论（5步标准蒸馏法，通用所有材料）
### 阶段1：素材预处理（去噪，第一步必做）
目的：剔除无效信息，统一素材口径，避免被表象干扰
1. **分类归档**
   按素材类型拆分：文字著作/演讲语录/对话访谈/案例决策/行为记录/观点评论。
2. **过滤杂质**
   剔除：临时情绪、客套话、场景化吐槽、跟风观点、单次偶然行为；
   保留：**反复提及的观点、一贯的选择、长期坚持的做法、对同一类问题的统一答案**。
3. **标记高频要素**
   圈出高频词汇、常用比喻、反复强调的关键词（词汇是思维的入口）。

### 阶段2：表层萃取——拆解「显性行为&观点」（看“他说了什么、做了什么”）
聚焦**具象输出**，搭建基础分析框架，用2个维度拆解：
#### 1. 观点维度：立场与判断
- 面对同一类问题，他**支持/反对/中立**的倾向是什么？
- 他评判事物的**显性标准**（好坏、对错、优先级）？
- 面对矛盾问题，他优先妥协还是坚持原则？

#### 2. 行为维度：动作与流程
- 解决问题的**步骤顺序**（先做什么、后做什么）？
- 信息获取习惯：优先看数据/经验/人情/直觉？
- 决策习惯：独断/协商/拖延/快速试错？

**工具**：制作「行为-观点对照表」，按问题场景逐条记录。

### 阶段3：中层萃取——提炼「思维框架&推理逻辑」（核心：思维模型）
从“言行”深挖**思考路径**，这是思维模型的核心层，分4个拆解角度：
#### 1. 归因逻辑（发生问题时，他如何找原因）
- 外归因：怪环境、他人、时机；内归因：找自身、方法问题；
- 单一归因 / 多层归因（是否会拆解根因）；
- 短期视角 / 长期视角。

#### 2. 分析框架（拆解事物的固定结构）
识别他惯用的**分类、分层、拆解模型**，常见信号：
- 总喜欢把事物分成2/3/N类（二分法、三段论）；
- 固定维度拆解：比如永远从「成本-收益-风险」「过去-现在-未来」分析；
- 复用经典模型（或自创模型）：金字塔、因果链、供需、系统论等。

#### 3. 推理方式（逻辑范式）
- 归纳法：从多个案例总结规律；演绎法：用固有规则推导结论；
- 线性思维（单向因果）/ 系统思维（多方关联、循环影响）；
- 具象思维（举例子、讲故事）/ 抽象思维（提炼概念、讲道理）。

#### 4. 决策模型（做选择的底层规则）
- 优先级排序规则：优先收益/风险/道义/效率/人情？
- 取舍原则：“两害取其轻”“长期价值优先”“底线至上”等；
- 容错观：厌恶风险 / 拥抱试错。

**输出成果**：整理出**可复用的思维模型清单**，每条模型配原文案例佐证。

### 阶段4：深层萃取——挖掘「观察视角&底层世界观」（最核心，决定独特性）
思维模型是“工具”，视角/世界观是“使用工具的人”，聚焦**不可言说的底层假设**，6个核心探究方向：
1. **价值排序**（终极偏好）
   自由、利益、道德、秩序、创新、安稳、情义……他心中的优先级是什么？这是视角差异的根源。
2. **人性假设**
   默认人性本善/本恶/趋利/中立？决定他看待人际关系、团队、社会的方式。
3. **世界假设**
   认为世界是**确定的/混沌的**、可预测的/随机的、零和博弈/共赢博弈？
4. **注意力焦点**（独特观察点）
   同样一件事：别人看结果，他看过程；别人看表象，他看本质；别人看局部，他看全局。**关注点=独特视角**。
5. **时间观**
   活在当下 / 紧盯未来 / 参照历史；看重短期快感 / 长期积淀。
6. **认知边界**
   他默认“哪些事是不可能的”“哪些规则不能打破”，也就是他的底层认知牢笼。

**关键技巧**：找**矛盾场景**——当利益、道义、现实冲突时，他的选择最能暴露世界观。

### 阶段5：整合验证&固化（去伪存真，形成最终成果）
1. **交叉验证**：用不同素材（不同时期、不同场景）核对同一结论，**反复一致的才是真认知**，单次特例直接剔除。
2. **反向推演**：用提炼出的模型/视角，反向预判他面对新问题的反应，验证是否成立。
3. **结构化输出**：分三部分固化成果：
   - 底层世界观（视角、信念、价值）
   - 核心思维模型（分析、推理、决策框架）
   - 典型应用场景+原文佐证

## 三、3个高效落地工具（直接套用模板）
### 工具1：三层蒸馏记录表（通用模板）
| 素材片段 | 表层（观点/行为） | 中层（思维逻辑/模型） | 深层（视角/底层信念） | 验证（多场景是否一致） |
|----------|------------------|-----------------------|----------------------|------------------------|

### 工具2：视角差异化对比表（凸显“独特性”）
把目标人物和普通人/其他同领域人物对比，快速找到独特观察角度：
- 大众关注点：XXX
- 他的关注点：XXX
- 差异背后的视角原因：XXX

### 工具3：关键词溯源法（极简入门法，适合短素材）
1. 统计全文**Top10高频词**；
2. 分析词汇属性：是偏向行动、情绪、逻辑、利益、理想？
3. 结合词汇，判断他的注意力方向和认知偏好。

## 四、进阶技巧&避坑要点
### （一）高阶萃取技巧（深挖独特视角）
1. **抓“隐喻与比喻”**
   人会用熟悉的事物理解世界，比喻直接暴露思维底色：比如总用“战场”比喻生活=竞争视角；总用“生态”比喻事物=系统视角。
2. **抓“默认前提”**
   很多观点不会明说前提，比如“做事要快”，默认前提是「机会转瞬即逝」，这就是底层假设。
3. **抓“批判与推崇”**
   他批判什么、推崇什么，直接定义价值取向。

### （二）常见坑（极易萃取失真）
1. **把临时观点当长期认知**：只看单次发言，忽略一贯言行 → 解决：多素材交叉验证。
2. **用自己的思维强行解读**：代入自身视角脑补 → 解决：只基于原文/事实，不主观臆断。
3. **混淆“方法论”和“世界观”**：思维模型是“术”，视角世界观是“道”，不要混为一谈。
4. **放大偶然行为**：一次情绪化选择≠固有思维 → 解决：以**多数常态**为准。

---

## 五、极简实操流程（快速上手版，适合新手）
1. 通读材料，标记**重复出现的观点、做法、用词**；
2. 回答3个问题：①他分析事情分哪几个维度？②做选择优先考虑什么？③看待人和事总盯着哪一点？
3. 总结：**他习惯用什么框架思考（思维模型）+ 他站在什么立场/关注什么（观察视角）**；
4. 用新场景测试结论是否成立，完成蒸馏。

# 蒸笼阁 · 写作向人物思维蒸馏 Skill
**Skill 标识**: zhenglongge-writing-distillation  
**版本**: v1.0  
**定位**: 专为文章写作场景优化的人物思维蒸馏工具。通过多源素材分层萃取目标人物的思考框架、独特观察视角与行文风格，输出可直接用于内容创作的标准化认知资产，解决写作中「观点平庸、视角雷同、文风失真」的核心痛点。

---

## 一、核心能力
1.  **多源素材智能分级**：自动识别素材类型与置信度，按写作价值权重分层处理，优先萃取高信度内容
2.  **五层认知蒸馏体系**：从表层文风到深层世界观逐层萃取，完整还原人物的写作逻辑与思想内核
3.  **写作特征定向拆解**：专项提取行文结构、论证范式、论据偏好、选题切口等写作专属维度
4.  **三重保真校验机制**：交叉验证+写作回测+边界校验，从根源避免语录堆砌、主观脑补与人设失真
5.  **可落地写作资产输出**：产出直接适配选题、立论、行文、润色全写作流程的标准化素材包

---

## 二、适用场景
- 深度人物稿、思想评论、行业分析类文章的创作
- 复刻特定学者、作家、行业大佬的观点与文风，撰写专栏/评论内容
- 借鉴多元人物的独特视角，提升文章的认知深度与内容差异化
- 批量拆解优质作者的写作逻辑，沉淀个人写作方法论
- 基于人物认知体系，生成系列化选题规划与内容框架

---

## 三、标准执行工作流
### 第1步：素材导入与分层归档
**目标**：去噪分级，建立高置信度的素材基底
1.  接收用户输入的全部文本素材，按「内容属性+置信度」双维度分类，写作价值由高到低分为四级：
    - **S级（核心素材）**：本人完整著作、系统专栏、亲笔长文、正式出版作品
    - **A级（重要素材）**：深度访谈全文、公开演讲逐字稿、正式媒体评论、署名文章
    - **B级（参考素材）**：社交媒体短文、随笔、零散语录、问答回复
    - **C级（边缘素材）**：第三方转述、他人解读、媒体报道（仅作背景参考，不进入核心蒸馏）
2.  自动清洗去噪：剔除客套寒暄、情绪性吐槽、临时语境发言、无关闲聊、口误修正内容
3.  对有效片段标注主题标签，形成可检索的素材片段库
4.  中间产物：分级素材库 + 主题化有效片段集

### 第2步：写作向表层特征提取
**目标**：拆解基础写作单元，建立观点与文风的原始数据库
1.  **观点表达单元提取**：对每一段有效内容，拆解为标准化三元组
    - 核心论点：人物明确表达的核心判断
    - 支撑论据：用于论证的案例、数据、典故、逻辑推导
    - 论证方式：立论/驳论、归纳/演绎、叙事说理/逻辑推演
2.  **文风表达单元提取**：统计并标记写作风格特征
    - 语言调性：犀利/温和、严谨/通俗、理性/感性
    - 句式节奏：长句/短句偏好、段落密度、转折节奏
    - 修辞习惯：常用比喻、典故、概念创造、金句范式
    - 高频词汇：核心概念、反复提及的关键词、专属术语
3.  同步标记：人物高频讨论的话题领域、偏好的案例类型
4.  中间产物：论点-论据-论证三元组库 + 文风特征标签集

### 第3步：中层蒸馏：思维模型与行文逻辑
**目标**：提炼可复用的写作骨架，对应文章的结构与分析逻辑
1.  **分析框架萃取**：识别人物拆解问题的固定维度与分类方法
    - 例如：永远从「成本-收益-风险」三维分析、习惯用「表层-中层-底层」三层拆解
    - 输出可直接用于搭建文章结构的思维模型，标注适用话题场景
2.  **推理范式萃取**：明确人物的核心论证逻辑
    - 归纳法/演绎法偏好、线性思维/系统思维、具象类比/抽象推导
3.  **行文结构萃取**：拆解人物常用的文章组织范式
    - 开篇模式：场景切入/观点直击/问题抛出/历史溯源
    - 展开逻辑：总分总递进、先破后立、并列分论点、层层深挖
    - 收束方式：总结升华、行动指引、开放式提问、回归本质
4.  **论据偏好萃取**：统计人物优先使用的论据类型
    - 商业案例/历史典故/个人经历/数据报告/思想实验/经典理论
5.  中间产物：核心思维模型清单 + 行文结构模板集 + 论证范式说明

### 第4步：深层蒸馏：独特视角与价值立意
**目标**：挖掘差异化认知，对应文章的灵魂与核心深度
1.  **注意力焦点识别**：提炼人物观察事物的独特切入点
    - 对比大众常规视角，明确人物「优先关注什么、忽略什么」
    - 例如：别人看结果他看机制，别人看收益他看代价，别人看个体他看系统
2.  **底层假设萃取**：挖掘人物不言自明的默认前提
    - 对人性、社会、商业、世界的基础信念，例如「人性本质是趋利的」「商业的核心是效率」
3.  **价值排序梳理**：明确人物评判事物的终极标准
    - 效率/公平、长期/短期、创新/稳定、理想/现实等核心价值的优先级
4.  **选题切口偏好**：总结人物切入话题的惯用角度
    - 宏大叙事/微观切口、追热点/谈本质、正面建构/反面批判
5.  中间产物：独特视角清单 + 价值立场图谱 + 选题方向库

### 第5步：边界补全：反模式与认知盲区
**目标**：明确写作禁区，避免人设失真与观点矛盾
1.  **明确反对的观点**：人物公开批判、否定的认知、做法与论调
2.  **价值观底线**：绝对不会认同、不会触碰的原则与立场
3.  **认知盲区**：公开承认不擅长、不讨论、不妄议的领域与话题
4.  **表达禁忌**：不会使用的语言风格、不会采用的论证方式、不会触碰的话题边界
5.  中间产物：边界与反模式清单

### 第6步：三重验证与写作资产封装
**目标**：保真校验，输出标准化写作资产
1.  **交叉一致性验证**：同一认知点需在≥2类不同素材、≥2个独立场景中重复出现，方可纳入核心资产库；单次出现的观点标记为「个人单次表达」，仅作参考
2.  **写作回测验证**：选取1-2篇人物公开发表过的完整文章，用蒸馏结果反向复刻，核心观点、行文逻辑、文风匹配度≥80%为合格
3.  **边界合规验证**：测试超纲话题与人物反对的立场，确认能主动识别并规避，不强行输出不符合人物认知的内容
4.  按标准格式封装为写作资产包，输出最终产物

## 四、核心输出产物规范
所有产物均围绕写作场景设计，可直接落地使用。

### 1. 《人物写作认知资产总报告》
核心交付物，按五层结构完整呈现，每条结论均标注对应素材来源与置信度等级。

| 层级 | 包含内容 | 直接写作用途 |
|------|----------|--------------|
| 文风表达层 | 语气调性、句式特征、修辞偏好、高频词汇、金句范式 | 语言润色，复刻人物写作质感 |
| 行文逻辑层 | 文章结构模板、论证方式、论据偏好、过渡手法 | 搭建文章骨架，组织论证链条 |
| 思维模型层 | 核心分析框架、推理范式、判断规则 | 搭建文章核心分析逻辑，输出深度观点 |
| 视角立意层 | 独特观察视角、价值排序、底层假设、选题切口 | 确定文章核心立意，打造差异化内容角度 |
| 边界底线层 | 反对观点、价值观底线、认知盲区、表达禁忌 | 规避写作雷区，确保人物认知一致性 |

### 2. 《写作调用系统提示词》
可直接复制给大模型使用的完整系统Prompt，支持一键让AI以该人物的思维方式与文风创作文章，包含：身份设定、核心认知约束、行文规则、边界要求。

### 3. 《核心观点与金句素材库》
按主题分类整理的核心论点、经典表述、原创概念，附带原文出处，可直接作为文章素材引用。

### 4. 《选题与立意参考清单》
基于人物视角延伸的可写话题、差异化切入角度，适配专栏评论、深度分析、人物稿等多种体裁。

### 5. 《行文结构模板集》
拆解人物最常用的3-5种文章结构模板，附带适用场景与使用说明，可直接套用写作。

## 五、质量约束硬规则
本Skill执行过程中必须严格遵守以下规则，不可突破：
1.  **无依据不输出**：所有结论必须有对应素材支撑，禁止主观脑补、过度解读与二次演绎
2.  **置信度透明**：所有认知点必须标注置信度等级，低置信度内容明确标注为「参考观点」
3.  **拒绝语录堆砌**：所有金句与观点必须归属于对应思维模型与视角，不得零散罗列
4.  **边界清晰**：必须明确标注认知边界，禁止将人物塑造成全知全能的形象
5.  **写作导向优先**：所有蒸馏环节最终指向写作可用性，避免空泛的理论化描述
6.  **差异化优先**：重点突出人物与大众认知的差异，避免输出普适性常识

## 六、使用建议
1.  素材越系统完整，蒸馏质量越高，优先提供著作、完整长文等S级素材
2.  素材覆盖场景越多元（不同时期、不同话题、不同体裁），越能还原真实的底层认知
3.  若有明确的写作体裁（如专栏评论、人物稿、深度分析），可在输入时说明，将针对性优化蒸馏方向
4.  支持多轮迭代：补充素材、调整重点后，可重新蒸馏优化结果
